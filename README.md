#ECG Baseline Removal: Processamento de Sinais Biomédicos 
---------------------------------------------------------
Este projeto implementa um filtro digital para remoção de ruído de baixa frequência (*Baseline Wander*) em sinais de Eletrocardiograma (ECG) reais. O código utiliza dados brutos do MIT-BIH Noise Stress Test Database (via PhysioNet) e aplica técnicas de Processamento Digital de Sinais (PDS) com Python.
